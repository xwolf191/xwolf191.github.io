总结

  • 初始容量为10
  • 动态扩容容量为指定容量或者2倍
  • 随机查找效率较高
  • 插入和删除慢,需要移动元素
  • 排序用归并排序和TimSort
  • 动态删除元素会抛出异常,可用迭代器实现
  • 线程安全
