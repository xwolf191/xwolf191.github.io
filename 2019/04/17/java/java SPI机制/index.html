Java 6 引入了一种功能，用于发现和加载与给定接口匹配的实现：服务提供者接口（SPI）。

概述

SPI 是 java 用于发现和加载与给定接口匹配的实现提供的一种实现机制。主要是为了扩展现有系统应用等。在 JDK 中也有大量应用,比如：

  • java.nio.file.spi.FileSystemProvider
