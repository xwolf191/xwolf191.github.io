用户线程和守护线程

发表于 2019-08-30 | 更新于 2020-07-19 | 分类于 java | 评论数：
本文字数： 8.6k | 阅读时长 ≈ 16 分钟

Java 提供了两种类型的线程：守护线程 和 用户线程。

  • 用户线程是高优先级线程。JVM 在终止之前会等待所有用户线程完成其任务。
  • 守护线程是低优先级线程。其唯一作用是为用户线程提供服务。

所有非守护线程执行完成后会退出 JVM,不管守护线程是否执行完成,所以在守护线程中不要做一些 IO 操作,可能会出现资源无法正常关闭等问题。
